# Ferienplanung – Frontend (React + Vite)

## Start

```bash
npm install
npm run dev
```

Standard-URL:

- `http://localhost:5173/`

## Build

```bash
npm run build
npm run preview
```

## Konfiguration

### `VITE_API_URL` (optional)

Wenn dein Backend nicht auf `http://localhost:3000` läuft:

```bash
VITE_API_URL=http://<server-ip>:3000 npm run dev
```
